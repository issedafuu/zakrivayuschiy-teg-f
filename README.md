# Закрывающий тег

Проектная работа курса «Вёрстка» от Яндекс Практикума.

## Описание

Сайт-дневник, в котором я рассказываю о своём пути в вёрстке. Восемь карточек — от первого знакомства с HTML до уверенности в своих силах. Каждая карточка сопровождается пиксельной иллюстрацией, личной историей и тегом-состоянием.

## Технологии

- HTML5
- CSS3 (flexbox, grid, адаптивность, медиазапросы, анимации)
- SVG-спрайты
- JavaScript (лайки, модальное окно)

## Особенности

- Резиновая вёрстка через `clamp`
- Фиксированный градиентный фон
- Сложная анимация лайка (сердце, искры, масштабирование)
- Модальное окно через `<dialog>`
- Пиксельные иллюстрации в едином стиле
